Introduction
This capstone project was completed as an internship at Elixir Software. The project focused on improving the user interface and user experience of a healthcare management platform by enhancing consistency, usability, and clarity across key system workflows.
IMAGE 1
Outcome
  • Improved UI consistency across forms, layouts, and components
  • Enhanced usability with clearer labels and confirmation messages
  • Improved administrative workflows and context visibility
  • Successfully deployed completed features into the beta environment
Goals
  • Improve UI consistency across the Elixir healthcare platform
  • Enhance UX by improving clarity, readability and ease of use
  • Improve key workflows by refining labels, layouts and confirmation messages
  • Ensure all UI changes integrate safely within the existing system without breaking functionality
Learning
  • Working with an existing industry codebase
  • GitHub workflows (branches, pull requests, conflicts)
  • UI/UX improvement in real-world systems
  • Agile / Kanban development practices
  • QA testing and iterative feedback cycles
  • Team communication in a professional environment
Methodology

The chosen methodology was a combination of Agile, Kanban, and Lean principles, used to manage development work in an iterative and flexible way within a real industry environment.


Key Features of These Methodologies

  • Iterative development with continuous improvement based on feedback
  • Kanban workflow used to visually track tasks through each development stage
  • Agile practices supporting regular communication, collaboration, and adaptability
  • Lean principles used to reduce unnecessary work and improve efficiency
  • Continuous testing and review cycles through QA and code review processes
IMAGE 2
Conclusion
The project successfully delivered UI/UX improvements for the Elixir healthcare management platform, improving consistency, usability, and clarity across key workflows. These changes have been deployed to production and are now available to clients.

This project demonstrates how focused UI improvements can enhance real-world systems while developing my skills in full-stack development, GitHub workflows, and industry teamwork.
